Hotel Reservation System
A database-driven hotel booking and management application using MySQL + Flask + HTML

Overview

The Hotel Reservation System is a backend driven application that automates hotel booking, room allocation, guest management, and payment tracking.
It eliminates the inefficiencies of manual reservation processes such as double booking, inaccurate records, and scattered dataâ€”by providing a centralized and real-time management platform.
The system uses MySQL for the database and Flask (Python) with HTML for the user interface.
It also implements core DBMS concepts such as ER modeling, relational schema, DDL/DML queries, triggers, functions, and stored procedures.

Features:
Admin login for secure access
Room booking with real-time availability
Guest management
Reservation tracking
Payment logging
Automated room status updates
Support for multiple hotels and room types
Full ER model and relational mapping
Stored procedures, triggers, and functions
Integrated Flask-based front end
